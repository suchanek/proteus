pdb2pov v1.23 

pdb2pov is a program capable of converting Brookhaven databank format atomic structure files to POV-Ray V2.x scene files. The program has a large number of scene options including:

  • CPK style rendering
  • Ball and Stick rendering
  • Automatic camera and light placement
  • Automatic area light placement for soft shadows
  • Generation of plain/checkerboard ground
  • Generation of plain/cloudy sky
    •  Version 1.23 adds additional parsing for the complete periodic table and a little smarter valence checking for sp2 atom rescaling. This keeps protons from getting buried by sp2 or aromatic ring systems (like benzene rings), leading to a nicer appearence. In addition, ball and stick atoms now automatically generate split bonds; i.e. the bond colors are chosen by the atom color, and each half bond uses its parental atom color.
